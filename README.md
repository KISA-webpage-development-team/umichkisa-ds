# @umichkisa-ds/web

KISA Design System — tokens and components for umichkisa.com.

## Scripts

| Script | Description |
|--------|-------------|
| `npm run build` | Compile JS and CSS to `dist/` |
| `npm run build:js` | Compile TypeScript only (tsup) |
| `npm run build:css` | Compile Tailwind CSS only |
| `npm run dev` | Watch mode for JS and CSS |
| `npm test` | Run tests (Vitest) |
| `npm run test:watch` | Run tests in watch mode |
| `npm run typecheck` | TypeScript type check without emit |
